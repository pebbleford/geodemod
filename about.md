# GD Mod Menu

A full-featured mod menu for Geometry Dash 2.2.

## Usage
Press **Tab** to open the mod menu in-game.

## Features
- **Noclip** - Pass through obstacles
- **Auto Play** - Automatically beats levels
- **Speed Hack** - Adjustable game speed
- **Instant Complete** - Complete any level instantly
- **Practice Music** - Normal music in practice mode
- **No Death Effect** - Disable death animations
- **Auto Retry** - Auto restart on death
- **No Particles** - Better performance
- **Unlock All** - All icons and colors
- **Safe Mode** - Prevents cheated score submission
- And more!
